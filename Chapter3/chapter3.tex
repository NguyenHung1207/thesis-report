\section{Tổng quan hệ thống}

Hệ thống là nền tảng web hỗ trợ kiểm thử tự động ứng dụng web. Người dùng có thể tạo và quản lý project, tạo test case từ mô tả bằng ngôn ngữ tự nhiên, quản lý dữ liệu kiểm thử, chạy test bằng AI agent hoặc replay script và xem kết quả thực thi.

Hệ thống cho phép scan/crawl website để thu thập sitemap và thông tin điều hướng khi cần. Đây là chức năng tùy chọn, không phải điều kiện bắt buộc để tạo project hoặc test case.

Trong quá trình thực thi, hệ thống ghi nhận log, screenshot, action history, verdict, error message và evidence. Các dữ liệu này được lưu để người dùng xem lại và phân tích nguyên nhân khi test thất bại.

\section{Nhóm người dùng}

Hệ thống hướng đến người dùng đã xác thực như QA/tester, developer hoặc người dùng nghiệp vụ có nhu cầu kiểm thử ứng dụng web. Trong phạm vi hiện tại, hệ thống chưa phân tách vai trò quản trị viên riêng.

Người dùng có thể thực hiện các chức năng sau:

\begin{itemize}
    \item Đăng ký, đăng nhập và đăng xuất.
    \item Tạo và quản lý project kiểm thử.
    \item Khai báo base URL và tùy chọn scan/crawl website.
    \item Tạo và quản lý test case.
    \item Sinh và quản lý test data.
    \item Chạy test bằng AI agent hoặc replay script.
    \item Chạy test với nhiều bộ dữ liệu.
    \item Quản lý test suite và object repository.
    \item Xem kết quả, log, screenshot, evidence và báo cáo lỗi.
\end{itemize}
