\section{Tổng quan hệ thống}
% =========================================================

Hệ thống là nền tảng web hỗ trợ kiểm thử tự động website. Người dùng có thể tạo và quản lý project, sinh test case từ mô tả bằng ngôn ngữ tự nhiên, quản lý dữ liệu kiểm thử, chạy test bằng goal-based agent (sau đây gọi tắt là agent) hoặc replay script và xem kết quả thực thi.

Hệ thống hỗ trợ scan/crawl website để thu thập sitemap và thông tin điều hướng khi cần. Đây là chức năng tùy chọn, không phải điều kiện bắt buộc để tạo project hoặc test case.

Trong quá trình thực thi, hệ thống ghi nhận trạng thái, log, screenshot, action history, verdict, error message và evidence. Các dữ liệu này được lưu để người dùng theo dõi kết quả và phân tích nguyên nhân khi test thất bại.

% =========================================================
\section{Nhóm người dùng}
% =========================================================

Hệ thống hướng đến người dùng đã xác thực như QA/tester, developer hoặc người dùng nghiệp vụ có nhu cầu kiểm thử website. Trong phạm vi hiện tại, hệ thống chưa phân tách vai trò quản trị viên riêng.

Người dùng có thể thực hiện các chức năng sau:

\begin{itemize}
    \item Đăng ký, đăng nhập bằng tài khoản hệ thống hoặc Google, khôi phục mật khẩu và đăng xuất.
    \item Tạo và quản lý project kiểm thử.
    \item Khai báo base URL và tùy chọn scan/crawl website.
    \item Sinh, chỉnh sửa và quản lý test case.
    \item Sinh và quản lý dữ liệu kiểm thử.
    \item Chạy test bằng agent hoặc replay script.
    \item Chạy replay script với nhiều dòng dữ liệu.
    \item Quản lý test suite và Object Repository.
    \item Xem kết quả, log, screenshot, evidence, report và phân tích lỗi.
\end{itemize}
